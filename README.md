# 🕒 Clock

A simple digital clock built with JavaScript, HTML, and CSS.

View the project on GitHub:
🔗 [https://github.com/mahak123569/clock](https://github.com/mahak123569/clock)

## 📌 About

This is a minimal **clock application** that shows the current time in real-time using JavaScript’s `Date()` object and updates the display every second.

Perfect for beginners learning DOM manipulation, timing functions, and basic styling.

---

## 🚀 Features

✔ Real-time updating clock
✔ Clean and simple UI
✔ Works in modern browsers
